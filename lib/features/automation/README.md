# lib/features/automation/

Feature module for task automation and scheduled AI-driven actions.

Provides UI and business logic for creating, managing, and running automated
tasks that can be triggered by schedules, events, or user-defined conditions.

## Planned Contents

- `domain/` — Entities (AutomationTask, TriggerCondition) and use cases
- `data/` — Persistence and scheduling data sources
- `presentation/` — BLoC, pages, and widgets for the Automation screen
