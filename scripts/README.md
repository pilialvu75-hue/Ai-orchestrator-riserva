# scripts/

Utility and automation scripts for development, build, and deployment tasks.

## Planned Contents

- `setup.sh` — Development environment setup script
- `build_android.sh` — Android APK build helper
- `build_windows.sh` — Windows executable build helper
- `download_models.sh` — Script to pre-download GGUF model files
- `lint.sh` — Run Flutter analysis and formatting checks
- `generate_build_readme_it.py` — Generate the Italian README/user manual on each build
