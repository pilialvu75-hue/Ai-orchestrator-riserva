# AI-Orchestrator-Core
Stable kernel repository for AI-Orchestrator.

This repository contains only the production-safe core:
- orchestration engine
- stable task execution system
- stable memory subsystem
- minimal safe AI routing/runtime integration

No experimental features are included here.
Experimental evolution (MobileIDE, prototype UI, advanced AI integrations, runtime experiments) belongs in AI-Orchestrator-Riserva only.

## Governance

Do not merge Riserva changes back into Core without explicit stabilization and review.
